4. Utilidad de Bootstrap vs CSS Puro

Usar clases de Bootstrap es increíblemente útil porque acelera el desarrollo. En lugar de escribir reglas CSS desde cero (como `display: flex;`, `justify-content: center;`, `padding: 20px;`, etc.), simplemente usamos clases predefinidas como `d-flex`, `justify-content-center` o `p-3`.

Esto no solo ahorra tiempo, sino que también garantiza la consistencia visual en todo el sitio y, lo más importante, nos da un diseño que se adapta a dispositivos casi sin esfuerzo. Es perfecto para prototipar rápidamente y para construir interfaces robustas sin tener que "reinventar la rueda" en cada proyecto.
